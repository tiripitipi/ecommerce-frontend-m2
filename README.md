# 🎾 El Rincón del Tenis

Tienda online de equipamiento de tenis. Proyecto del Módulo 2 del Bootcamp.

La temática la elegí porque el tenis me parece un deporte que me encanta y quería que el proyecto tuviera algo de mi propia personalidad.

## ¿Qué tiene el proyecto?

- Página principal con los productos disponibles
- Página de detalle para cada producto
- Carrito de compras que recuerda lo que agregaste aunque cierres el navegador

## Tecnologías usadas

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript vanilla

## Estructura del proyecto
```
ecommerce-frontend-m2/
├── index.html
├── detalle.html
├── carrito.html
├── css/
│   └── styles.css
├── js/
│   └── main.js
└── img/
```

## Cómo verlo

Clona el repositorio y abre el `index.html` con Live Server en VS Code.
```bash
git clone https://github.com/tiripitipi/ecommerce-frontend-m2.git
```
